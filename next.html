question-1: What are the different ways to select element an in DOM?
The different ways to select an element in the DOM: 1. getElementById() – Select by ID, 2. getElementsByClassName() – Select by class, 3. getElementsByTagName() – Select by tag, 4. querySelector() – Select by CSS selector.
Question-2: What is the different between InnerHTML, InnerText, and textContent?
The difference between innerHTML, innerText, and textContent: 1.innerHTML – Gets or sets HTML inside an element, 2.innerText – Gets or sets only visible text (ignores hidden text) , 3.textContent – Gets or sets all text, including hidden ones.
Question-3: Wht is event delegation in the DOM?
Event delegation is when an event listener is added to a parent element to manage events for child elements. It uses event bubbling to handle dynamic elements.
question-4: What is the bubbling in the DOM?
Bubbling in the DOM is when an event starts from the target element and propagates upwards through its parent elements. This allows parent elements to catch events triggered by their child elements.
